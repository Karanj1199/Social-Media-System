# Social Media System

A full-stack social media platform built with React, Spring Boot, and PostgreSQL.

## Tech Stack
- React
- Spring Boot
- PostgreSQL
- Spring Security
- JWT
- WebSocket
- Redis
- Docker

## Day 1 Status
- Backend project created
- Frontend project created
- PostgreSQL connected
- Base architecture finalized

- User profiles
- Posts
- Feed
- Likes and comments
- Follow/unfollow
- Real-time chat
- Notifications
- Search/autocomplete
- Redis caching/rate limiting
- Dockerized deployment
